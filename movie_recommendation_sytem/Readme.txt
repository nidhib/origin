Project is on topic - Movie Recommendation System
Skills used are  - Vanilla JavaScript, HTML and CSS
Project is user friendly and its main aim is to recommend a movie name based on 
        *his interest in Genre,
        * the time he has to watch movie (Small amount of time, medium or long time duration) and
        * His mood type like happy, sad, feeling light and so on.


Once, the end user chooses all the three options, the result is displayed accordingly as Movie name in a message form.
